# Speaker Module

## Brief Introduction

![](./images/render_speaker.png)

Speaker module is a mp3 player controlled by serial intruction. Put sound files to in and control it with controller module.

## Specification

Size: 48 x 48 x 11.6 mm

Charging Power: 1W

Supported File: mp3

Memory Space: 16MB
